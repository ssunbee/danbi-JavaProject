# 📋Java GUI 이용한 회원관리 만들기
기존에 있는 소스코드를 재활용하여 콘솔화면에서 사용되는 회원가입 프로그램을 만들어 보았습니다.

## 🔧개발환경
+ intelliJ 2022.09.30 ~ 2022.10.7
+ Java(JDK 1.8)
+ GitHub
+ Windows 10

## 📋사용 기술  
### 백엔드
+ JAVA 

## 📋주요 기능
+  정보입력
+  정보삭제 
+  정보검색
+  정보수정 
+  회원 목록보기
+  종료
